◆このソフトについて

Windowsでたまに発生する、無音状態から何かしらの音を再生し始めたときに
フェードインするような効果を無効にするソフトです。

実際やっていることは非常に単純で、無音の音を延々と流しているだけです。


◆インストール方法

特に操作は必要ありません。zipファイルを展開すれば完了です。


◆操作方法

DisableFadein.exeを実行してください。

実行中、タスクバーに表示されるアイコンを右クリックすることで
機能の一時停止・再開ができます。


◆アンインストール方法

フォルダごと削除してください。レジストリやAppDataは使用していません。


◆動作環境

.NET Framework 4.8がインストールされているWindows PCで動作します。

Windows 10 Pro バージョン21H1で動作確認しています。
